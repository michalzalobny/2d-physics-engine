# 2d-physics-engine

2D Physics engine built from scratch in C++. Based on the [course from Gustavo Pezzi](https://pikuma.com/courses/game-physics-engine-programming)

# Run commands

- **With custom script**

  - `./build-and-run.sh`

- **With Makefile**

  - `make build` or just `make` (it will pick first command)
  - `make run`
  - `make clean`
